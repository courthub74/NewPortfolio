About.

User centric developer that designs and develops productivity web applications as well as applications aimed at helping budding web developers gain valuable work experience.

5 years ago I realized that I liked to make tasks easier for people through code and development when I helped the backend developers at my first tech support job. They developed a stress test application with an interface that was very confusing to use.

I took to learn HTML and CSS on my own. In turn, I made the whole experience more user friendly for myself and coworkers. After that, I decided to teach myself and work in the craft of web development.

While teaching myself to code, I've also done freelance dev jobs under my brand CourDevelops.

Things I'm passionate about:

The User Interface: A web app should lead the user with an easy, thoughtless experience.

The Gig Economy: People should be able to leverage technology to obtain work and/or experience without the burden of human bias or logistics.

Blockchain/Web 3: The ability to exchange value instantly on the web is so necessary and paramount going into the future.
